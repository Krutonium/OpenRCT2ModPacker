# OpenRCT2ModPacker
Mod Packer for OpenRCT2Launcher

Allows simple Packing and Unpacking of mod packs.

YOU MUST LAY OUT YOUR FILES LIKE THEY WILL BE IN THE INSTALLED GAME:

-Tracks/ShuttleLoop.TD6  
-Scenarios/LeafyLake.SC6  
-Data/CSS50.Dat  
-ObjData/Object.Dat  
-Saves/Save.SV6
